# Order Book Imbalance Backtester

A quantitative trading project focused on market microstructure analysis and order book imbalance.

## Features

- Real-time Binance order book data
- Order book imbalance calculation
- Trading signal generation
- Risk management
- Backtesting engine
- Performance metrics

## Technologies

- Python
- Pandas
- NumPy
- Requests
- Matplotlib
- Binance API
